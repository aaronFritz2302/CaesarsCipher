CaesarsCipher Documentation¶

Introduction¶

This is a simple python package for encrypting and decrypting text in Caesar’s Cipher.

Note

This documentation was generated on Aug 18, 2021. Version 0.0.1

Installation¶

Install the CaesarsCipher package using PIP

pip install CaesarsCipher

Usage¶

How To Use The CaesarsCipher package

 1# Import The Ceaser Class From The CeasersCipher Module
 2from CaesarsCipher import Caesar
 3
 4# Encrypt Your Text Using Ceaser.encrypt('your_text_here',shift_value)
 5print(Caesar.encrypt("Aaron Fritz",10))
 6
 7# If You Don't Specify Shift Value In The Function For Encryption, The Default Shift Will Be 10
 8print(Caesar.encrypt("Aaron Fritz",10))
 9
10# Decrypt Your Text Using Ceaser.decrypt('your_text_here',shift_value)
11print(Caesar.decrypt("Kkbyx Pbsdj",10))
12
13# If You Dont Know The Shift Value, Decrypt Your Text Using Ceaser.decrypt('your_text_here') for Brute Force Decryption
14print(Caesar.decrypt("Kkbyx Pbsdj"))
